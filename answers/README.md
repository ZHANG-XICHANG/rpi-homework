# 答案區 (Answers Area)

這個目錄存放所有作業的標準答案和參考解答。

## 答案命名規則

每個答案應該與對應的作業編號相匹配，格式為 `answer-XX.md`。

例如：
- `answer-01.md` - 第一次作業的答案
- `answer-02.md` - 第二次作業的答案
- `answer-03.md` - 第三次作業的答案

## 答案內容應包含

1. 對應的作業編號和標題
2. 詳細的解答步驟
3. 程式碼（如果適用）
4. 解釋說明
5. 常見錯誤提示

## 使用說明

學生在完成作業後，可以到這個目錄查看標準答案，對照自己的解答。

**注意：** 建議學生先獨立完成作業，再來查看答案。
